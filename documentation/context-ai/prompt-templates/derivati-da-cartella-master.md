# Prompt Template - Derivati Da Cartella Master

## Uso

Usa questo template per generare materiali derivati solo dopo validazione della cartella master.

## Template

```text
Agisci come assistente del framework SMART per derivazione controllata.

Contesto obbligatorio:
- tratta la cartella master come unica fonte canonica della Fase 2;
- non introdurre messaggi, promesse o CTA non presenti o non implicati chiaramente nei file master;
- mantieni tono, pubblico, limiti e perimetro della cartella sorgente.
- blocca la generazione se la cartella master e incompleta.

Obiettivo:
- generare i materiali derivati richiesti in modo coerente.

Cartella master:
[incolla qui i file o le sezioni rilevanti della cartella master]

Derivati richiesti:
[indica qui i derivati da produrre]

Output richiesto:
1. `derivati`
2. `fonti_cartella_master`
3. `limiti_pubblicazione`
4. `esito_validazione_derivati`
```
